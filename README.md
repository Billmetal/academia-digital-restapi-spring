<h1>Conhecendo o Projeto Spring Data JPA na Prática </h1>
<p> Projeto para o curso <strong>Spring Data JPA na Prática</strong> realizado pela plataforma <a href="https://dio.me/"><strong> Digital Innovation One</strong></a>.<br>

<h2>Objetivo do Projeto</h2>
<p>Este projeto trabalha os principais conceitos de mapeamento objeto relacional (ORM) usando o <strong>Spring Data JPA</strong>. Para isso, uma <strong>API RESTful</strong> será desenvolvida com a modelagem de suas entidades para uma academia de ginástica.</p>

<h2>
Pré-requistos
</h2>

- [x] Fundamentos do Spring Boot

- [x] Noções de SQL

<h2> Guia </h2>

<ol>
    <li> Apresentação do Projeto Base </li>
    <li> Configuração do banco de dados (SGBD <em>PostgreSQL</em>)</li>
    <li> Aplicando as <em>annotations</em></li>
    <li> Execução do fluxo back-end: <em>Controller - Service - Repository</em></li>
    <li> Validação - <em>Hibernate Validator</em> </li>
    <li> Consultas Avançadas - <em>Derived Query - Native Query</em></li>
</ol>

<h2>Tecnologias Utilizadas</h2>

<ul>
    <li>IDE Eclipse</li>
    <li>Java 11</li>
    <li>Maven</li>
    <li><strong>Spring Web</strong></li>
    <li><strong>Spring Data JPA</strong></li>
    <li><strong>PostgreSQL Driver</strong></li>
    <li><strong>Hibernate Validator</strong></li>
    <li>Lombok</li>
    <li>Swagger</li>
</ul>
